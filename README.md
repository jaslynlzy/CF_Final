# Cirencester Foodbank Food Voucher Distribution

Interactive Streamlit dashboard to help Cirencester Foodbank optimise food voucher distribution, featuring choropleth maps and per capita postcode analysis. Developed for Cirencester Foodbank as a Data Science Consultant at 180DC Bristol.
